# camera-demo
H5调用摄像头的demo

## step 1
运行Application

## step 2
访问 http://localhost:9001/camera.html

## step 3
访问 http://localhost:9001/client.html

## step 4
点击 shot 按钮

## 主要实现

### 网页 camera.html
拥有摄像头的终端，打开 camera.html 可以调用、开启摄像头

### 网页 client.html
拍摄控制终端，打开 client.html，点击 shot 按钮，可以使 camera.html 的摄像头进行拍摄
